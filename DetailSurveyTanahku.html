    Redesign User Experience and User Interface of SurveyTanahku Apps.

    Freelance

    Jul 2022 - Nov 2022

    Mobile Apps

    SurveyTanahku is an application for land mapping and field surveys belonging to the ATR / BPN ministry. This application must be able to operate online and offline. Previously there was the SurveyTanahku version 1, my job was to redesign version 2 of SurveyTanahku. Of course with the new features.

    Play Demo

    Problem i want to solve.
    There are 3 main problems, 1. Old application design that doesn't refer to UI and UX principles, I have to improve the design. 2. The feature structure of the application must be rearranged to make it more concise. 3. I have to learn new terms in the field of civil engineering to be able to understand the application requirements properly.

    My Role In the Project, Who Else in the project and My responsibility.
    My role in this project is as a UI/UX Designer... actually my work is more of a UI Designer, namely updating existing application designs but I still have to add a UX "touch" to new applications. In this project i worked with 3 Android developers and 1 Project manager.

    A challenging and interesting problem for me.
    the challenge in this project lies in the function of the application and how this application will be used... a new experience for me in designing applications for civil engineering people in the field. The problem I have is understanding new terms and new disciplines, in this case civil engineering. For a good design, at least I have to understand it right?

    The Solutions i gave.
    therefore I learned to understand new technical terms in order to understand how the application works.... then I designed a new interface (following the latest UI trends) and followed the applicable UX laws in designing the SurveyTanahku application.

    The impact of my final solution.
    unfortunately, the project that has been running for 4 months has to go into a pending period due to internal problems, so I can't see the impact of the design I made.

    The lesson that I took in this project, namely work as a professional. In this project I learned to design and work in a structured and professional manner because I worked with PMs and Developers who are very experienced in their fields.
